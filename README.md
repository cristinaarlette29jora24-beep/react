# React + TypeScript - Bootcamp InfraOps

Módulo 3 del bootcamp. Componente genérico DataTable construido con React, TypeScript y Vite.

## Tecnologías
- React 18
- TypeScript (strict)
- Vite
- date-fns

## Cómo ejecutar

npm install
npm run dev

Abre http://localhost:5173

## Estructura

src/
├── components/
│   └── DataTable.tsx   
├── utils/
│   └── fechas.ts       
└── App.tsx
